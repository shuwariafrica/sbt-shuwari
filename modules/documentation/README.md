# sbt-shuwari

[![Build Status](https://github.com/shuwariafrica/sbt-shuwari/actions/workflows/build.yml/badge.svg)](https://github.com/shuwariafrica/sbt-shuwari/actions/workflows/build.yml)

Nothing to see here. Internal sbt plugin collection for Shuwari Africa projects.

## Modules

| Artifact             | Coordinates                                                  |
|----------------------|--------------------------------------------------------------|
| `sbt-shuwari`        | `"africa.shuwari.sbt" % "sbt-shuwari" % "@VERSION@"`         |
| `sbt-shuwari-core`   | `"africa.shuwari.sbt" % "sbt-shuwari-core" % "@VERSION@"`    |
| `sbt-shuwari-mode`   | `"africa.shuwari.sbt" % "sbt-shuwari-mode" % "@VERSION@"`    |
| `sbt-shuwari-header` | `"africa.shuwari.sbt" % "sbt-shuwari-header" % "@VERSION@"`  |
| `sbt-shuwari-scalac` | `"africa.shuwari.sbt" % "sbt-shuwari-scalac" % "@VERSION@"`  |

`sbt-shuwari` bundles `sbt-shuwari-core`, `sbt-shuwari-header`, and `sbt-shuwari-scalac`.
